# SmartStudy AI Assistant

SmartStudy AI Assistant is a full-stack AI-powered study tool designed for computer science students. It provides topic-based explanations, structured responses, and quiz generation through a chat-style interface.

## Features
- AI-powered study chat assistant
- Topic-based learning support
- Structured responses with simple explanation, technical explanation, and practical example
- Quiz generation by difficulty
- Chat history persistence
- Clear chat function
- Responsive UI

## Tech Stack
- React
- Vite
- Flask
- SQLite
- Groq API

## Supported Topics
- Python
- Databases
- Algorithms
- Networking
- Artificial Intelligence
- Object-Oriented Programming
